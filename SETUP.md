This file explains how to take this template and set up your new flutter x firebase project


1. Go to firebase console and create a new project
2. Set it on the blaze (pay-as-you-go) plan
3. Go to build -> storage, set up storage with the free location (US-central1)
4. Go to the terminal and run firebase init
5. Select use existing project, select your new project
6. Select firestore database, functions, and storage in the list of things to include
7. Finish the setup and run firebase deploy
